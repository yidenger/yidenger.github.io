node.js全局模式指的是模块通过-g参数安装的，否则为本地模式

node.js本地模式与全局模式的区别

　　　模式　　　 　　　可通过require使用　　　 注册Path

全局 否 是

全局包安装位置：/usr/local/lib/node_modules/ , 在文件中使用require获取并不会搜索此路径，
npm link 命令可以解决此问题(BTW, window不支持此命令 /(ㄒoㄒ)/~~).

  • 本地模式：把包作为工程运行一部分
